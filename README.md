# 🏪 Corralones - App de Contacto

App web para que una persona mayor pueda contactar corralones fácilmente por WhatsApp.

## Características

- 📱 Diseño optimizado para celular (letra grande, botones grandes)
- 💚 Un click para contactar por WhatsApp
- ✓ Se marca automáticamente como contactado
- 📍 Filtro por zona geográfica
- 🔄 Botón para reiniciar contactos
- 📲 Se puede instalar como app en el celular

## Cómo usar

1. Abrir en el celular: `https://TU_USUARIO.github.io/corralones-app/`
2. Elegir la zona
3. Tocar el botón verde de WhatsApp
4. ¡Listo!

## Despliegue en GitHub Pages

1. Crear repositorio en GitHub: `corralones-app`
2. Subir los archivos:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/corralones-app.git
   git push -u origin main
   ```
3. Ir a Settings → Pages → Source: main branch
4. Esperar 2 minutos y listo!

## Datos

Los datos de los corralones están en `index.html` y se actualizan desde Google Sheets.
